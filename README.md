# script de processamento de logs

Tutorial de desenvolvimento de script para monitorar os logs do sistema.

## Preparando o ambiente
Você precisará ter um ambiente Linux, como o WSL 2 no Windows.

Caso utilize algum sistema operacional baseado em Linux, como o Ubuntu ou o macOS, ou use uma máquina virtual com alguma distribuição Linux, não é necessário instalar o WSL 2.

## Projeto

Abra o terminal, e crie um diretório chamado myapp/logs

```bash
mkdir -p myapp\logs
```

Entre no terminal com o comando:

```bash
cd myapp\logs
```

Nesse diretório crie um arquivo chamado myapp-backend.log

```bash
touch myapp-backend.log
```
